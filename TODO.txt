Questão: Desenvolvimento de Sistema de Suinocultura


DONE
Você deve criar um sistema web para gerenciamento de suinocultura que atenda aos requisitos especificados abaixo.

Requisitos Funcionais:

DONE
Entrada de Ração:

Crie uma página que permita a entrada da quantidade de ração recebida.
Armazene essas informações no banco de dados.
Distribuição para Matrizes:

DONE
Crie uma página que permita a distribuição da ração para as matrizes, indicando a quantidade a ser fornecida para cada matriz.
Atualize o estoque de ração no banco de dados.
Distribuição para Berçário e Machos:

DONE
Crie páginas distintas para a distribuição de ração no berçário e para os machos, com a capacidade de inserir a quantidade específica para cada um.
Atualize o estoque de ração no banco de dados.
Controle de Estoque:

DONE
Mantenha um controle de estoque atualizado no banco de dados.
Relatório Diário:

DONE
Crie uma página que exiba um relatório diário com a quantidade total de ração fornecida e a quantidade restante no estoque.
Semáforo de Entrada de Ração:

DONE
Implemente um semáforo na página de entrada de ração que mude de cor conforme a quantidade de ração inserida, indicando visualmente se a quantidade é suficiente (verde), moderada (amarelo) ou insuficiente (vermelho).
Requisitos Técnicos:

DONE
Use HTML, CSS e JavaScript para a construção da interface.
